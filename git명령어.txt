1.초기화

git init  

2.원격저장소 추가

git remote add origin https://github.com/shan25kit/ajax.git 

3.원격저장소 확인

git remote -v

4.현재 브랜치 확인

git branch

5.개별 브랜치로 변경

git checkout [개별브랜치명]

6. 개별 브랜치에 최신코드 가져오기

git pull origin develop



1. git push -u origin [개별브랜치명]

2. git remote set-url origin https://shan25kit@github.com/shan25kit/ajax.git

명령어 입력 후 github 로그인을 하라고 하면 브라우저로 로그인 하시면 됩니다.

git fetch origin
