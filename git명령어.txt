1.초기화

git init  

2.원격저장소 추가

git remote add origin https://github.com/shan25kit/ajax.git 

3.원격저장소 확인

git remote -v

4.현재 브랜치 확인

git branch

5.개별 브랜치로 변경

git checkout [개별브랜치명]

6. 개별 브랜치에 최신코드 가져오기

git pull origin develop



1. git push -u origin [개별브랜치명]

2. git remote set-url origin https://shan25kit@github.com/shan25kit/ajax.git

명령어 입력 후 github 로그인을 하라고 하면 브라우저로 로그인 하시면 됩니다.

git branch -vv

git fetch origin 


git remote set-url origin https://shan25kit@github.com/shan25kit/ajax.git

git branch- vv 해서 develop이랑 개별 브랜치 뜨는거 확인 후 

git pull origin develop으로 코드가져오시면 됩니당!


방법1 같은 브랜치를 노틉북과 데스크탑에서 사용
원격저장소 연결
git clone 원격저장소 주소
git checkout [개별브랜치명]
git pull origin [개별브랜치명]

중요: 작업 환경 바꿀때마다 pull 해서 꼭 현재 최신 상태 코드 받아오기!!! 

방법 2 데스크탑 테스트용 새로운 브랜치 만들기
git checkout [개별브랜치명]
git checkout -b [개별브랜치명-desktop-test]
필요시 테스트용 브랜치 메인 브랜치에 병합
git checkout [개별브랜치명]
git merge [개별브랜치명-desktop-test]
git push origin [개별브랜치명]
